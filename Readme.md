# Node.js basics

scripts for tasks execution are in 'exec' folder
use `npm run <script-name>` and the following scripts to run
- fs:copy
- fs:create
- fs:list
- fs:read
- fs:remove
- fs:rename
- cli:parse-env
- cli:parse-args
- hash:calculate-hash
- streams:read
- streams:write
- streams:transform
- zip:compress
- zip:decompress
- wt:workers
